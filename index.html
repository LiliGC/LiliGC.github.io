Aquí hay algunos proyectos que he realizado, principalmente usando el framework Django:

Blog

Aplicación web de un blog basado en el tutorial de Django Girls, realizado en parte en el workshop virtual Django girls Bristol en el que participé, cuenta con la opción de realizar post, editarlos, publicarlos, eliminarlos. También se puede dejar comentarios a los post. Pudiendo moderarlos el propietario del blog autorizándolos o eliminándolos.

Laboratorios LGC

Aplicación web de un sitio de venta de productos de laboratorio. Cuenta con templates para crear cuenta, iniciar sesión, formularios de contacto, ver tablas con clientes, productos, profesionales, pudiendo editarlos, o eliminarlos al tener permisos exportando datos a pdf, excel o imprimirlos. El catálogo de productos cuenta con carrito de compra.
